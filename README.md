# NeuroLift AI

NeuroLift AI is a powerful and intuitive application designed to revolutionize your fitness journey. Whether you're a seasoned athlete or just starting, our AI-powered features provide personalized workout plans, detailed progress tracking, and insightful nutritional guidance to help you achieve your goals faster.

## Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1fdJEphsCUFNIFtv78mv_5LmZojx0Dm3f

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Create a `.env.local` file by copying the `.env.example` file and set the `GEMINI_API_KEY` to your Gemini API key.
3. Run the app:
   `npm run dev`
